
# UNO No Mercy

Videojuego de cartas inspirado en la versión UNO No Mercy

## Descripción

Este proyecto consiste en una adaptación digital del juego de cartas UNO No Mercy, una versión más intensa y estratégica del UNO clásico. El juego incluye cartas especiales, acumulación de penalizaciones y mecánicas avanzadas como intercambio y rotación de manos.

El desarrollo se realiza utilizando Python y la librería Pygame, implementando una interfaz 2D interactiva y un sistema de juego por turnos contra oponentes controlados por la computadora.

## Características planeadas

- Sistema de turnos
- Cartas numéricas y especiales
- Mecánica de acumulación de robo de cartas
- Cambio de color y comodines
- Intercambio de manos (7)
- Rotación de manos (0)
- Inteligencia artificial básica
- Interfaz gráfica en 2D
- Sistema de monedas y ligas

## Tecnologías utilizadas

- Lenguaje de programación: Python 3
- Librería: Pygame

## Estado del proyecto

Actualmente el proyecto cuenta con:

- Sistema de turnos funcional para 1 jugador y 3 bots
- Jugador principal interactivo
- Bots automáticos básicos
- Validación de jugadas por color y valor
- Robo de cartas desde el mazo
- Carta central dinámica
- Detección de victoria
- Indicador visual del jugador en turno
- Cambio de dirección (Reverse)
- Bloqueo de turno (Skip)
- Penalizaciones de robo (+2, +4, +4 Reverse, +6 y +10)
- Mazo ampliado con cartas numéricas, de acción y comodines de UNO No Mercy
- Interfaz gráfica inicial en 2D
